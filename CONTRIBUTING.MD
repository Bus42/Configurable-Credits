# Contributing Guide for the Project

For new features, bug fixes, or improvements, please follow these steps:

1. Clone the repo to your local machine
2. Create a new branch for your changes
3. Make your changes and commit them with a clear message
   1. Keep your commits small and focused
   2. Use descriptive commit messages that explain the purpose of the change
4. Push your changes to GitHub
5. Open a pull request against the development branch of the original repository

I will review your changes and provide feedback. If everything looks good, I will merge them into the testing branch. Once the changes are tested and approved, they will be merged into the main branch.

If you have any questions or need help, feel free to reach out to me on [Discord](https://discord.gg/396GaEYSV7) or [Twitch](https://twitch.tv/legen_dary42) or open an issue in the repository.
